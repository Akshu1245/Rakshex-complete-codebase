# VS Code extension

Package: `apps/vscode-extension`.

## Status: Beta

- Secrets via VS Code `SecretStorage`
- Commands: scan workspace / changed files
- Marketplace packaging scripts under extension `PUBLISHING.md`

## Local development

```bash
cd apps/vscode-extension
pnpm install
pnpm test
# F5 in VS Code Extension Development Host
```

Do not claim Marketplace install counts unless verified from the publisher portal.

## First Comment (founder post — post immediately on launch)

Hey Product Hunt! 👋

I'm Akshay, and I built DevPulse because I kept making the same expensive mistake: shipping AI agents that quietly burned through API budget or leaked credentials I forgot were in a collection.

**The problem:** When you're building fast with LLMs and APIs, security and cost issues hide in plain sight — inside Postman collections, environment configs, agent scaffolding. They don't surface until something breaks or a bill arrives.

**What DevPulse does:** It lives in your VS Code sidebar and continuously watches for:
- Leaked API keys and credentials in collections
- Hidden token costs from unoptimized LLM calls
- Security misconfigs in AI agent frameworks (LangChain, CrewAI, AutoGen)
- Shadow APIs your agents call that you've forgotten about

**How it works:**
1. Install the extension (free)
2. Connect your API key (generate one at devpulse.in)
3. Import a Postman collection or OpenAPI spec
4. First findings in 60 seconds

**Privacy:** We scan API metadata and configs — your source code never leaves your machine.

We're in public beta and genuinely want feedback. What does your AI agent stack look like? Happy to answer any questions below 🛡️
